open source big data integration, analytics, and visualization platform

What are you doing with your data?

There's tremendous insight hiding in your existing data. Do you have the tools to discover and exploit it? We didn't, so we decided to build them. At Altamira, we're leading the creation of an open source big data integration, analytics, and visualization platform. Basically, it's a way to aggregate your data, organize it, and extract useful insights.

We need your help and guidance! We've just released Lumify into the wild and it's still a young project. We're looking for people interested in kicking the tires, providing feedback, and helping to improve the project. If you're interested, please read the docs and check out the source.

Analyze relationships, automatically discover paths between entities, and establish new links in 2D or 3D.

Overlay data as layers on a map for a geographical view of the data model. Pluggable map providers allow integration of corporate GIS solutions.

Organize work into separate workspaces that can be shared with colleagues. Updates are pushed to everyone viewing the workspace in real-time.
